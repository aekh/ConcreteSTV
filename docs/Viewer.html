Instructions for viewing election changes

This page allows you to examine specific vote changes that can change outcomes in elections.

Click on the browse button above to select the .vchange file. You will be shown one or more small vote changes or vote additions that alter the outcome in that contest, specifying:

  • Ballots: how many votes are involved
  • Effect: which candidate(s) are added or removed from the list of elected candidates
  • How: What the change involves. Changed means some ballots were changed; Added means some votes were added. Removed means some ballots were removed. ATL means some above the line votes were affected (this can be prohibited in some scenarios). First Preferences means some first preference votes were affected (this can be prohibited in some scenarios). Affected Verifiable means some votes considered verifiable were affected (this is situation dependent). Direct to beneficiary means some manipulations directly went to some candidate (or their party in an ATL vote) who ended up winning as a result of the manipulation. Direct from victim means some manipulations directly came from some candidate (or their party in an ATL vote) who ended up losing as a result of the manipulation. Generally, the fewer tags in this column the harder it is to detect.
  • Details: a summary of the vote changes. Hover your mouse over them to see exactly which votes are changed or added
