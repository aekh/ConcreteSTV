This table lets you choose some "major parties" or candidates. Each formal ballot paper is then assigned to one of the "major parties" based on which of these gets the first preference for one of them. If none, it is considered exhausted. This determines the appropriate column for the ballot. The ballots are also divided based on the first preference. This determines the appropriate row for the ballot. Each cell in the table contains the number of ballots for that row and column. The sum of each column is the "major party" preferred vote for that party; the sum of each row is the number of first preference votes for that candidate (or party).
